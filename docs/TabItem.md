# kaiui-tab-item 

The `<kaiui-tab-item>` component. 

- **author** - Sebastian Baar 
- **license** - MIT 

## slots 

- `default` Use this slot to include components in a tab-item. 

## props 

- `name` ***String*** (*required*) 

  The Name 

- `selected` ***Boolean*** (*optional*) `default: false` 

  If the Tab Item should be selected
