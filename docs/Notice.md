# kaiui-notice 

**This component is automatically integrated as a Mixin. Just use**

`this.showNotice("ion-battery-empty", "Battery Full", "Battery is fully charged")`

**in your components.** 

- **author** - Sebastian Baar 
- **license** - MIT 

## data 

- `icon` 

  The Icon CSS class
  ***String*** (*required*)
  `default: ''` 

**initial value:** `''` 

- `title` 

  The Title
  ***String*** (*required*)
  `default: ''` 

**initial value:** `''` 

- `subtitle` 

  The Subtitle
  ***String*** (*required*)
  `default: ''` 

**initial value:** `''` 

- `time` 

  The Time in ms
  ***Number*** (*optional*)
  `default: 4000` 

**initial value:** `4000`
