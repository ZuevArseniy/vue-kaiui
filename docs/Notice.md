# kaiui-notice 

**This component is automatically integrated as a Mixin. Just use**

`this.showNotice("ion-battery-empty", "Battery Full", "Battery is fully charged")`

**in your components.**

- `title`

- `subtitle` 

**initial value:** `''` 

- `time` 

**initial value:** `4000`

- `show(icon, title, subtitle, time)`
